# HR Employee Attrition Analysis 📊

## 🏢 Business Case & Project Overview
Employee turnover is an incredibly expensive challenge for modern organizations. This project analyzes the **IBM HR Analytics Employee Attrition dataset** (1,470 employees, 35 features) to identify the primary drivers of turnover and discover which departments/demographics face the highest flight risk.

By connecting a local relational database infrastructure directly to an interactive BI semantic model, this project delivers data-driven retention strategies designed to mitigate talent loss and reduce company recruitment expenditures.

### 🎯 Key Questions Addressed:
1. What are the top 5 factors predicting employee turnover?
2. Which departments and job roles exhibit critical attrition thresholds?
3. How do compensation bands and tenure milestones correlate with employee departures?

---

## 🛠️ Tech Stack & Architecture
*   **Database Engine:** MySQL Server (Ingestion, structural exploration, data cleaning)
*   **BI & Analytics Tool:** Power BI Desktop (Data modeling, DAX measures, executive dashboarding)
*   **Data Pipeline:** Live relational mapping using the MySQL Connector/NET pipeline (Import Mode)
